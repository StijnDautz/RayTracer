BONUS IMPLEMENTATIONS
Anti-aliasing:
Send more rays in the direction close to the ray's direction and average these colors

Soft shadows:
Send more shadow-rays from points close to the intersection point and average the shadow value for a 
more 'soft' looking shadow.
